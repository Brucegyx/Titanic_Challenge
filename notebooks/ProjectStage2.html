1.2.a Data Transformation¶

  • Due to the skewed data and minimizing effect from outlier (mean = 29, max = 80), fill in the empty 'Age' entries with the median of the age instead of mean.
  • Despite the high missing rate of column 'Cabin', it could provide information related to a passenger's representational group. A cabin number is a combination of a passenger's deck label followed by a number. Hence, we can separately encode their deck labels while missing entries will be assigned a 0.*
  • Map 'Sex' values to integers (male: 0, female: 1)
  • Map 'Embarked' values to numerical values ('S': 0, 'C': 1, 'Q': 2)

*There should be 7 decks with passengers on Titanic according to https://www.encyclopedia-titanica.org/cabins.html

titanic_df['Embarked'] = titanic_df['Embarked'].map({'S': 0, 'C': 1, 'Q': 2})

1.3.b How Features are used by the Models¶

Random Forest: features will be sampled to build each decision tree. Each decision tree will split its nodes using the bootstrap sampled input data with mean squared error to evaluate the split. The result will be the average of all decision tree results

Logistic Regression: calculated a weighted sum of features then apply sigmoid function to produce a likely result.

1.3.d Model Estimation Results¶

The two selected models are trained with stratified sampling and 5-fold cross validation

Since the evaluation metric used by Kaggle for this dataset is accuracy. We will use the average accuracy of 5 folds to determine which model performs better. We will also use t-test and significance level of 0.05 to determine if one model is significantly better than the other.

In [ ]:
from sklearn.model_selection import KFold, StratifiedKFold
from sklearn.ensemble import RandomForestClassifier
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

rf_val_accaracies = []
lr_val_accaracies = []

# Split data with stratified cross validation into 5 fold
skf = StratifiedKFold(n_splits=5, shuffle = True, random_state = 42)
for i, (train_index, val_index) in enumerate(skf.split(X, y)):
  X_train, X_val = X.iloc[train_index], X.iloc[val_index]
  y_train, y_val = y.iloc[train_index], y.iloc[val_index]


  # Initialize and train a Random Forest Classifier
  rf_classifier = RandomForestClassifier(n_estimators= 100, random_state=42)
  rf_classifier.fit(X_train, y_train)

  # Make predictions on the validation set
  rf_predictions = rf_classifier.predict(X_val)

  # Evaluate the Random Forest model
  rf_accuracy = accuracy_score(y_val, rf_predictions)
  rf_precision = precision_score(y_val, rf_predictions)
  rf_recall = recall_score(y_val, rf_predictions)
  rf_f1 = f1_score(y_val, rf_predictions)
  rf_val_accaracies.append(rf_accuracy)

  print(f"\n Fold {i}, Random Forest Classifier Accuracy: {rf_accuracy}")


  # Initialize and train a Logistic Regression Classifier
  lr_classifier = LogisticRegression(random_state=42)
  lr_classifier.fit(X_train, y_train)

  # Make predictions on the validation set
  lr_predictions = lr_classifier.predict(X_val)

  # Evaluate the Logistic Regression model
  lr_accuracy = accuracy_score(y_val, lr_predictions)
  lr_precision = precision_score(y_val, lr_predictions)
  lr_recall = recall_score(y_val, lr_predictions)
  lr_f1 = f1_score(y_val, lr_predictions)
  lr_val_accaracies.append(lr_accuracy)

  print(f"\n Fold {i}  Logistic Regression Classifier Accuracy: {lr_accuracy}")


# calculate the average accuracy of 2 models
rf_avg_acc = np.mean(rf_val_accaracies)
lr_avg_acc = np.mean(lr_val_accaracies)
print(f"\nAverage Accuracy of Random Forest Classifier: {rf_avg_acc}")
print(f"Average Accuracy of Logistic Regression Classifier: {lr_avg_acc}")

# define a data preprocessing function
def preprocess_data(df):
  df['Age'] = df['Age'].fillna(df['Age'].median())


  df['Cabin'] = df['Cabin'].fillna('U0')
  df['Cabin'] = df['Cabin'].apply(lambda x: x[0] if x is not None else 'U')

  df['Cabin'] = df['Cabin'].map({'A': 1, 'B': 2, 'C': 3, 'D': 4, 'E': 5, 'F': 6, 'G': 7, 'T': 8, 'U': 0})

  df['Sex'] = df['Sex'].map({'male': 0, 'female': 1})

  df['Embarked'] = df['Embarked'].fillna(df['Embarked'].mode()[0])
  df['Embarked'] = df['Embarked'].map({'S': 0, 'C': 1, 'Q': 2})

  df = df.drop(['Ticket', 'Name'], axis=1)

  df = df.drop('PassengerId', axis=1)

  scaler = MinMaxScaler()

  numerical_features = ['Age', 'Fare', 'SibSp', 'Parch']
  df[numerical_features] = scaler.fit_transform(df[numerical_features])
  return df

1.5 Conclusions and Learning¶

Conclusions¶

The target of this project is using ML models to predict whether a passenger can survive in the Titanic incident based on demographic data. After careful data analysis and processing, two models that are common for classification are evaluated on theri accuracy.

Although the logistic regression and random forestmodels do not have significant perforamnce difference on this task, we selected the random forest model based on its slightly better average accuracy. Then, from the feature importances of the trained random forest model, we can determine that Age, Fare, and Sex features are the 3 most impactful ones to determine passengers' survival.

The trained model has an accuracy of 75.1% on unseen dataset. For all the passengers predicted as 'Survived' by the trained model, the distribution of the 3 most important features highlight that female, age between 20 and 40, ticket fare lower than 100 are more likely to survive than others.

In the contrary, social class is not considered as important as Age, Sex, Fare. The distribution of 'pclass' features among all predicted survivors is not very skewed towards any particular social class.
